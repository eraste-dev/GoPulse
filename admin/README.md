# GoPulse Admin (Future Phase)

Ce dossier est réservé à l'application d'administration centralisée.

## Objectifs
- **Gestion Multi-tenant** : Permettre de gérer plusieurs clients ou organisations sur une même instance.
- **Gestion des Comptes** : Interface pour créer, modifier et réinitialiser les accès des utilisateurs.
- **Supervision Globale** : Vue d'ensemble de tous les agents déployés.

## Stack Prévue
- Technologie similaire au dashboard (Next.js / Prisma) pour une cohérence maximale.
- Système de Rôles (RBAC) : SuperAdmin, Admin, Viewer.

*Note : Le développement de cette partie débutera dans la prochaine phase du projet.*
